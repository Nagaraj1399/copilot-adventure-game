# GitHub Copilot - Build Your Own Application

This repository contains three starter templates to help you learn GitHub Copilot by building simple applications.

## 📚 What's Included

### 1. Python - Command-Line Adventure Game
**File:** `python_adventure_game.py`

A text-based "Choose Your Own Adventure" game built in Python. Navigate through a mysterious forest and make choices that affect your journey.

**How to run:**
```bash
python python_adventure_game.py
```

**Use GitHub Copilot to enhance it:**
- Add more game paths and story branches
- Implement a scoring system
- Add difficulty levels
- Create a save/load game feature

---

### 2. JavaScript/Node.js - Text-Based Space Adventure
**File:** `adventure_game.js`

An interactive space station mystery game built with Node.js. Solve puzzles and make decisions to escape a malfunctioning space station.

**How to run:**
```bash
node adventure_game.js
```

**Use GitHub Copilot to enhance it:**
- Add inventory management
- Implement puzzle solving mechanics
- Add time-based challenges
- Create multiple endings based on choices

---

### 3. Portfolio Website - HTML/CSS Template
**Files:** `portfolio/index.html` and `portfolio/styles.css`

A responsive portfolio website template showcasing projects and skills.

**How to view:**
Open `portfolio/index.html` in your web browser.

**Use GitHub Copilot to enhance it:**
- Add JavaScript interactivity
- Create a functional contact form
- Add animations and transitions
- Implement dark mode toggle
- Add project filtering

## 🚀 Getting Started with GitHub Copilot

1. **Open the repository in VS Code or GitHub Codespaces**
2. **Install GitHub Copilot** (if not already installed)
3. **Open any file and start coding**
4. **Press `Tab` or `Ctrl + Enter` to accept Copilot suggestions**

## 💡 Tips for Using Copilot

- Write clear comments describing what you want to do
- Start with function signatures or function names
- Use type hints (Python) or JSDoc comments (JavaScript) for better suggestions
- Feel free to modify and customize the suggestions
- Experiment with different prompts to see how Copilot responds
